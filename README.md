# Note-pedia

## Introduction 

A basic frontend-only note-taking app where users can write, store, edit, and delete their personal notes. 
Try the app [here](https://note-pedia.netlify.app)

----

## 🚀 Key Featues

- Persistent storage of notes (using browser's Localstorage)
- Ability to write notes in Markdown language
- Ability to create, add, edit and delete tags for each note
- Searching notes by title/relevant tag
- Fast, simple, and free to use

----

## ⚙️ Tech stack

- React
- Typescript
- Bootstrap CSS

----

## 🛠️ Setup on your local machine

Clone the repository and run:
<pre>npm install</pre>
This installs all dependencies.

Then run:
<pre>npm run dev</pre>

Open *localhost:5173* in your browser to see the app running
